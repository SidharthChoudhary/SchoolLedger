Bulk Import Payroll Data

Instructions
  • Upload a CSV file with historical payroll data to populate salary records.
  • Required columns: Emp_ID, Month, Payable_Salary
  • Optional columns: Old_Dues, Other_Amount, Note, Manual_Work_Days, Manual_Leave_Days
  • Emp_ID: Must match an existing employee number (e.g. 1000)
  • Month: Format YYYY-MM (e.g. 2024-04). Session is auto-detected.
  • Payable_Salary: Final salary amount paid (numeric, e.g. 8000)
  • Old_Dues / Other_Amount: Additional due amounts (default 0)
  • Manual_Work_Days / Manual_Leave_Days: Leave blank if attendance register is filled
  • Use the Dry Run option to preview changes before importing.
  • Choose how to handle duplicates: Skip existing records, Update them, or treat as Error.
