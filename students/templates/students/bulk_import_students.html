Instructions
  • Upload a CSV file with student data.
  • Required columns:
  • First_Name, Last_Name, Gender, Fathers_Name, Mothers_Name, Class_Code, Session
  • Optional columns:
  • Class_Name, SRN, NIC_Student_ID, Date_of_Birth, Admission_Date, Transport_Method, RTE, Primary_Account_Holder, Fathers_Phone, Mothers_Phone, Gardians_Name, Gardians_Phone, Previous_School, Medical_Conditions, Dietary_Restrictions
  • Gender values: male, female, 3rd_gender
  • Date format: YYYY-MM-DD recommended
  • Boolean values: yes/no, true/false, 1/0
  • Class_Code should match an existing class.
  • Session should match an existing session label.
  • Use dry run first before final import.
